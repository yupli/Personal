# 23.九月开学篇(1)

> 子昱 闲谈之余 2023年10月1日 15:33 湖北

原来我从四月份开始就断了记录的习惯。原来已经过了五个月。

每次说着，要重拾，要重拾。却总有无数的理由强拽着我，诉说你现在没时间。

其实是本性的慵懒。

复盘会给人带来新的思考，新的启迪，新的方向。希望你重新启航。

想先聊聊情感。挥之不去的话题，萦绕。

首先，我必须对自己坦诚：对她的爱慕之意，现在仍然未消散。

就像她说的，她慢热。当我想离她远点，控制距离的时候，我似乎涌上了她的心头。可此时我好像不能正对着她，总怀有一丝芥蒂，明明我们关系很好。她单向的输出此时没有接受到及时的回应，便也冷却下来了。

现在的我，不知该如何形容那时，是惋惜吗？可明明当时我已经下定决心了，原来我还是那样优柔，忐忑。

如今，新的一个学期又到来，经过暑假的洗礼，我俩还是有很多的事情一起干。小组作业，比赛，我还是习惯想和她一起完成，而她也从不会拒绝我。不时地，我甚至会有一些嫉妒心理，那是因为她与其他男生开始走得近。我始终没有做到和她"正常"相处。

日日顺的比赛让我们有了一次疯狂的机会——夜爬武功山。

习惯性地和她走得很近，我一直逼问自己，"为什么要如此？"。其实是心里得到没有一个确定的回答。

弥漫的大雾将我们紧紧包裹，不见前方，不闻声响。无边寂静让人幸福得要晕倒，彻底沉睡于此。

那时，外围空气冰冰凉，心里滚烫烫。但我没什么话能讲的，只能细微地感受一切。

登上山顶，我们选了弱风之处休憩，讲了很多神志不清的话。被喊CP时的感受，是微甜的。等到她回话，我才敢顺着她继续说，生怕出现意外之词。原来，那时的她对我的玩笑话放在心上了，有些愠气。因为，直至天亮、下山，她对我的眼神总是不对劲的。

下山之后的我，又傻了吧唧地追问她，企图同她解释清楚。含糊表达、逆天发言，让我进一步陷入窘境。下火车那一刻时，我的心紧缩到不敢呼吸。随后，她说，她认为我一直都很了解她，知道她是怎么样的人。我对这句话，感到一种莫名的滞空。

之后的对话缓缓走向了正常，但我有意引其到山顶的那个问题"你们为什么不谈恋爱呀？"，企图得到她的看法，说出我的回答。

——我想得到一种解放——我想彻底断开这段连接——心无旁骛地

她的一系列话解开了我原来的某些疑惑，我也向她给出了我的回答。

我想告诉她，最简单的一个理由，就足以把我们分开。

恋人于她而言，只是关系很好的一个朋友。她说，她很悲观，不相信文学中那些浪漫和理想在现实落地。

我则与之不同。若不怀揣着些玫瑰和晚霞，世界会变得枯燥和无聊。

其实，我对她的回答表示怀疑，一个悲观的人，竟能表现出如此多的热情与幼稚可爱。

我，一个根本算不上乐观的人，平日里也没有太多的热烈表现。或许，人就是五光十色，异样纷呈的。

昨日寻我名字，问我手围，又让我心生溢彩，甚至可以说得上是狂喜。欢愉的同时，也让我疑惑。我还是不懂她。她的性格也不会与我直说。

这算是纠缠吗？ 这种"陪伴"有意义吗？

同班的我们注定在这两年是要继续走下去的。我还是抱着最坏的打算，希望用最少的念头，等着毕业的到来。

100天的承诺我会做到的。算是我的一份坚持，依托。
