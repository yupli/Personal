# 记一个雨夜

> 子昱 闲谈之余 2023年10月2日 23:49 湖北

今天是2023年10月2日
是我的农历生日

她给我送来了礼物和祝福
心里自是万般感动

夜晚十点
她说要保持距离为好
我说好

心里是什么滋味呢
是难过
平静的低落
弥散开来

她说得并没有什么错
我也答应的很好
只是这种突如其来直杀心口

我们之后还有很多事要一起完成
每日还是得见到
这其实才是最难熬的

我们本来没什么深仇大恨
一段微妙的关系惹得两人都心生沉闷
或许这是情的奇妙之处

我有所预感
之后的日子不会真的僵化陌生
又是很多七上八下
我很担心我受不受得了

时间其实很快的
毕业就在明朝
彼时你又该是何感受呢

我认为
根源还是我自己
对恋爱的看法与他人有异
往后的生计往往是第一个浮现脑海的

再过个五年十年
可能我早已找到另一半
这里的一切会随云烟消散的
——这几乎是必然事件
可又如何呢
预知了未来的结果并不能减少现在的痛楚
就像知道死亡的节日必将降临
却还在为生的历程烦忧

写下这些并不能减少心里的难过
但记录
本身意义非凡
以后的我会从中看到现在的我
高中时的随笔在现在看来便是如此

无论如何呢
明天会到来的
我只能对明天的我说一句，加油
加油呢
排解郁结看来是人生一大难题
苦笑
