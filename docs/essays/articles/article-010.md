# 书后思

> 子昱 闲谈之余 2022年12月24日 23:53 江西

一个晚上一本书 《第七天》

人很轻松，很自在，读完之后是一种奇妙的体验。

首先，其实读书是一件很惬意，也很轻易的事情。随手一拿便开始看了，读毕，深度的思考就涌上脑海，多么享受！这远比电子榨菜和游戏要令人愉悦得多。不从任何其他目的来讲，娱乐形式本无差别。但代入主观体验之后，还是会不自觉得认为，读书是一种高贵的体验！高贵的享受！

回到这本书，读完《第七天》给我的印象是：奇妙，荒诞，现实，虚幻

处处是现实，但处处虚幻

暴露了各种问题，但也展现了人性中的美好，感情的美好。

余华说，写书时，是带着这些感动写下去的。是啊，尽管各种丑陋层出不穷，可是很难不被那些画面打动！一幕幕的真实，心疼极了！

我依然很珍贵这种感情的存在，它就在我们身边，所以定要好好把握。不同于那个时代的苦命与颠沛，我们的幸福来的太降临，太自然了。一时间，幸福已在每天每天与我们打招呼。
