# 听国奖答辩后

> 子昱 闲谈之余 2023年10月8日 21:34 湖北

现在心情已经平复了，没了刚才骑车时的百般复杂。

看见同学站上答辩舞台，感受着你与他们共同经历的一点一滴，你也喜悦万分，为他们录屏，拍照。

但是结果总不能照顾到每一个人，有人欢喜有人低落，如同上次一样。

离开会场，脱离人流，心里涌现了复杂的滋味。光彩夺目的他她们，难免会让人生出些许自卑。昊双同学答辩的最后一番话让我印象深刻，大意是，今天能站在这里已是荣幸，我们也许会惊叹于他人的精彩耀眼，对自己所做过的一切开始产生怀疑，可你要知道人生会各自发光的，不必太在意比较。

我很佩服她能在讲台上如此讲，也许只是想博得观众评委的认同。但此番话击中了我。我原以为到了大三的我，成绩还算稳定，心里对考试分数、相互竞争已经没了太多的波澜，我的好胜心应当没那么强了。但我还是错了，其实是我主观想弱化它。实际的它砰砰地在跳动。

弱化好胜心的原因之一，是大家都在努力前进，相较于他们，自己没达到更好的目标，但又不想让自己太过于难过低落。这便是手段吧。

说到这里，方才骑车的情绪又跑了回来。这种隐含的难过，在心里藏了好久。我做不到完全消除它，因为它是人性的一部分，是我的一部分。

自己给自己的宽慰不可少：

如果总是在比较，这世上比你厉害的人不可数。哪怕你在华科占优，倘若在清华北大，在更高的学府，你又会如何？对吧，比较是永无止境的。永远保持热爱和活力才是生活幸福的源头。

这可能是我需要终身追寻的了。

（写这篇时，也正和国奖得主分享喜悦吗，安慰落选的邓，同时想到还有更多的他她在积极生活，在默默努力。）

每个人都在发光呀，我为什么要独自抑郁呢？热爱可抵岁月漫长，愿我找到心之所爱！

高潮和低谷交替登场
欢喜和悲伤此起彼伏
收获的背后是无声的一点一滴的积累

想要享受百分之一的鲜花，就要有百分之九十九的汗水。
