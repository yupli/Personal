# 十月：不停歇的脚步，自在的路

> 原创 子昂 闲谈之余 2024年10月24日 19:02 新加坡

九月与十月恍然过隙，保研的旅程也告一段落了。

大四生活已在不知不觉中开启了。保外的好处在此时骤然显现，环顾四周，有同学已经早早进组，提前成为研究生er，为导师前赴后继，忙碌与酸甜，皆在不言之中；也还有人在为考研考公努力，每日的规律作息，复习计划，他们的压力也只有自己能体会；或有同窗在焦头烂额，为今天的工作面试不佳而感到懊悔，亦或是刚拿到一份比较满意的offer。真心希望身边的朋友都有一个好的归宿。

是幸运，也是坚持，给了我一份闲暇宝贵的时光。

轻松了下来，于是开始为自己的大四填充色彩，铺上层层石子。成为了老师的RA，和向师兄一起写论文，顺便钻研毕设。这些便是目前大四生活的主线，充盈着我大部分的时间，在无形中催促着我前行。

主线之外，则是多样的"生活"。是娱乐，是放空，是自我。

在管院忙碌论文时，我可以放下手上的任务，和朋友们一起去K歌，也可以准时观看激动人心的世界赛，尽享美妙时刻；从图书馆的座位上站起，一边放松腰腿，一边悠闲地穿梭于不曾光顾过的书架，随意翻看起一本感兴趣的闲书；走出东十二，是安静的枫叶被风一片片刮到我跟前，裹紧衣裳，把手塞进口袋，我漫步走过无人的韵体小路，享受宁静安详。

此时此刻在东九的窗边，有所想，于是打开公众号，开始写。这是属于我的自由，属于生活的自在。

在多样的生活中，想给自己提几个目标。

第一件事，每周锻炼的频率保持在2~3次。身体是革命本钱，往后的日子我和我的身体还要并肩作战很长时间。我把大四当作是健体养生的一年。

第二件事，规律作息。虽稍空闲，但不可纵欲过度。每日的精神状态是最好的映照。这两件事我将其摆在最前面，也是因为健康在首，才不辜负。

第三件事，阅读与写作的坚持。我已经讲过多遍，内心也明白其重要性，但容易被各项娱乐活动掩盖而去。在睡前的时段，在碎片化的时间，请多多读书；每月的总结请别忘记；有感而发的随笔，它等着你。

大四。仍是一段脚步不停歇的时光。与此同时，希望它是一条自在的路。

# 八月：快反馈与慢反馈

> 原创 子昂 闲谈之余 2024年8月26日 18:06 湖北

暑假的进度条已然近末，整整八月，我几乎都在家中，享受，放纵，将未来之事抛之脑后。这与我原本的计划相差无几。

来到学校之后，我尝试地问自己一个问题，我过的快乐吗？思考之后，给出的回答是，相当快乐。这种快乐源自无拘束的娱乐，无担忧的空闲。我可以恣意地躺在床头，刷一整天的手机；或是坐着电脑桌前上分。视频、游戏等娱乐形式的快反馈机制，使得我沉溺其中，不愿出来。

但弊端在于，当你回顾过去这段时间的收获，是游戏上的虚拟分数和回忆不起来的视频内容。讲这些弊端并非是单纯批判这些事物，纵乐享受从来不是错误。对于苦命的人而言，这些是无聊生活的重要调剂。回到我本身，长时间的陷入其中产生事后的虚无感——时间快速流逝但一无所获的感觉。再加上窝在家中不锻炼，二者叠加，使得快乐感变得单薄易碎。

相比以上，书籍、吹笛等慢反馈的事情显得不太有吸引力，因为你明白做这些事情不会让多巴胺快速产生，在做的过程中需要一定的自律性，还要想办法给自己正反馈，以此坚持下去。在这个暑假中，我并没有做到这一点，但我几乎没有后悔。开头我就说了，这与我原本的计划相差无几。

如果将这些慢反馈的事情坚持下去，聊以数月，甚至几年，内心丰盈，收获颇多，此时会见证"岁月待人好"。困难之处是始终如一，不轻言放弃。

我之所以提到快反馈和慢反馈这两类事物，是想为自己探索一种生活方式。在未来的研究生生涯，工作生涯，我大概率成为不了"强自律型"的人。人生苦短及时行乐的主张我是比较赞同的，但另一方面我又想通过书籍等需要长期投入的事物来获得不一样的"能力"。这种能力可能是对自我、对世界的深刻认识，也可能是掌握一项难度很大的技术。

总之，我认为生活是一味汤，它既需要一些看似无用的短期行乐，也需要持之以恒的长期投入。前者好比是白糖，后者是中药。如果觉得生活太太苦，不妨加点甜；若是甜齁过了头，不妨吃吃苦。

# 24.7月随记

> 原创 子昂 闲谈之余 2024年7月5日 21:28 湖北

刚刚思忖良久，该如何回复乔...

前两天，乔打了一则电话过来，询问我暑假是否有时间做家教，酬金丰富。我心动了，于是答应把空闲时间发给他。忙了两天，重新思考这个事情，我却犹豫了。

其一，7月的夏令营安排紧凑，让我没有办法腾出多余的时间和精力来做别的事情。到了8月份预推免报名阶段，虽说正式考核要等到9月份，但是材料、笔试、面试的准备也许让我心乱如麻，无暇顾及其他。这两个月决定人生求学之路的下一站会前往何方，重要性不亚于高考。我是一个不喜欢让自己后悔的人，对于未知的未来，我尽力奔赴局部最优解，不想落下遗憾。

其二，8月份的大段空闲时间，我想留给自己。两三个星期的时间，若是用来做家教，想必会收获颇丰。不可置否，金钱的力量无穷，是独自在外求学的底气，让我在花销时少几分顾虑。但是转念一想，现阶段的我非要这笔钱不可吗？

竹笛计划、想看的书、去外婆家、和兄弟姐妹再聚......

这些都是我预想可以在暑假做的事情，是支撑我走到现在的动力之一，是惬意，是美妙，是遐想。有人说，长大成人就意味着要走进社会现实，直面经济现状。这固然没错，但并不意味着，要将为数不多的闲暇时间全部交付给物质提升。今后有太多的日子要努力，要赚钱，要顾家，何差这一时。迈入研究生阶段之后，再无寒暑假，要应对科研、实习、工作等压力。学生时代的"休息"将一去不复返，往后是成人谈论不尽的生活议题。

所以，为什么要把自己早早埋葬在劳碌中。忙碌的目的是为了理想的生活，而不是忙碌本身。前段时间高考结束，有学生询问准大一该如何利用好这个暑假。我与一些博主的想法不谋而同，那就是好好的玩，尽兴的玩。当我去玩耍，去享受生活，我才能体会到目标感，我才知道我真正喜欢什么样的生活，然后收拾行囊再次出发。

我所畅想的日子，是工作与生活大部分分离。做有挑战性，有意义的工作，尽自己所能为公司创造价值，相应地，公司给予我可观的酬劳。下班之后，做自己喜欢的事情。阅读喜欢的作家最近出版的小说，逗逗自己可爱的小猫，和爱人一同去公园散步，记录下最近的所思所想，如果有必要可以赚赚外快......

畅想的生活总是带着美好温馨的滤镜，但人还是要有"理想型"的，万一实现了呢。

讲到这，想必该如何回复已经很明晰。

# 记第一次夏令营结束

> 原创 子昂 闲谈之余 2024年6月14日 12:22 湖北

正上着大数据分析的课，一条邮箱消息弹了出来，是清华的拒信。好吧，其实也是预料之中的事，但是她进了，还是出乎我的预料的，连忙向她贺喜。短暂平复了一下心情，想继续写完作业，却发现眼前一片茫...

有很多同学也来关心我的情况，自然只能回复"没有捏"，心中的失意被悄悄掩盖在文字下。终于下课了，带着笑意，拉着同学几个去吃牛肉粉。坐在后座上，漫不经心地回答着孙的问题，心里却是思绪万千。

"自己的失败固然成功，但朋友的成功更令人揪心"，孙说。我露出了苦笑。是啊，我心里其实很难过，但下意识地想克制，不示于人。直面内心，我为什么难过？有落选的失意，有没发挥好的不甘心，还有鲜明对比之下的黯然。可能最后一项在我心中激起的水花是最大的。

对名校光环的向往，那条阶梯忽地一下消失了，于是我摔了下来，疼痛无比。我想想我为什么痛：有一小部分嫉妒成分，难以避免但要坦诚承认。可嫉妒使人面目全非，这句话我不敢忘。这一小部分的嫉妒其实站不住太久，我真心地希望她去往更好的地方。大部分的痛都来源于对比，或者说"对比感"。直白的说，我会在意他人的目光，尽管这些目光中有一些是我假想出来的。否定感会随之出现，低人一等？技不如人？不管怎样，我坦率地接受心里的负面情绪，这些就是我第一时间会想到的。是啊，我会不自觉地和他人相比，况且还是她。当然，这是人生重大修行问题之一，以后的我必然对此不会陌生。

一个人无法总处于较高的位置，享受着某种"轻松"，体会不到其他人的"迷惘"。至少对于我来讲，我希望我学会珍惜当下的"轻松"，努力争取未来的"轻松"，做好面对"不轻松"的准备。另一方面，不要忘了理解他人的"迷惘"，他们是另个时空的你，你只是更幸运一些。今日的失意估计还是会蔓延一段时间，但请你坚持走下去，你也会收到一捧属于你的鲜花！但也请你记住现在的难过，未来的你还有很多很多这样的时刻。永远不要忘记鼓励自己，我从来都相信你。另外，在身边人失意时也请拉他一把，他也是你。

以上是我最直接的感受。感受过后，是理性的思考。

我真的适合读博士吗？这个问题已经飘荡在我的脑海中好几个月了。想想博士的生活，工作与生活的交融，reading和writing的反复，忍耐与不安，坚持与坚定。毫无疑问，博士的5，6年正好是人生最美好最珍贵的年华，如果怀着热爱，将其奉献给科研，我想这是人生幸事。但倘若不呢？可能面临的是，折腾了3，4年，却没有任何成果。毕业压力，就业压力，扑面而来。就业之后，非升即走也是一道必须跨过的大河。

所以，往好的方向想，拒信其实是帮我自动排除了一个选项，一个我十分犹豫的选项，说不定是一件好事。可能有点自我安慰的感觉哈哈哈。当然，与梦校失之交臂的遗憾会一直存在。对于这条我没有机会踏足的迷人小径，以后的我可能会幻想，当初我走了这条路会发生什么呢......不管怎样，捉摸不透的激流已经将推向了我推向了另一条航道，我能做的是将小船的舵牢牢控制，驶向我心中的港湾。
