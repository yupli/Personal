# 《我与地坛》最后一句的理解

> 子昱 闲谈之余 2024年2月8日 00:28 江西

宇宙以其不息的欲望将一个歌舞炼为永恒。这欲望有怎样一个人间的姓名，大可忽略不计。

这句话读了好几遍，依然没太读懂史铁生想要传达的确切含义，可能每个人的理解不一样吧。

宇宙运转不息，即是永恒。这永恒的运转其实也极具美感（想象一下无穷无尽的星球在永不止息地运转），称之为歌舞也就不难理解了。在某种程度上，这可以理解为宇宙的欲望？（当然是以拟人的角度，人总是喜欢把事物拟人化）。

人间之于宇宙，显得无比渺小。我想，史铁生将视野放置于宇宙的角度时，是否也感觉到人间的一切其实显得沧海一粟，无足轻重。

每每遇到不幸，或者心情苦闷时，我总是尝试以这种方式来舒缓内心的消极。对于宇宙而言，我们其实也没那么重要，一切显得都非常普通，甚至可以说，一切都是没有意义的。也许，在数万亿光年之外的某个星球之上，有一个智慧生命，也发出了同样的感慨。最后，你们都将归为尘埃。而这个过程，对于整个宇宙而言就是弹指一挥间。所以，知道这些以后，苦难的阴影是否消减了些许呢...

高中时第一遍读完《三体》就有如此感触，大学寒假又酣畅淋漓地读了第二遍。今天看到这句话，我突然发觉，其实很多人同我一样，也曾这么想过。今天特有感慨，将其写下。

——我想得到一种解放——我想彻底断开这段连接——心无旁骛地

其实，我对她的回答表示怀疑，一个悲观的人，竟能表现出如此多的热情与幼稚可爱。

这算是纠缠吗？ 这种"陪伴"有意义吗？

同班的我们注定在这两年是要继续走下去的。我还是抱着最坏的打算，希望用最少的念头，等着毕业的到来。

100天的承诺我会做到的。算是我的一份坚持，依托。

身体健康永远是最宝贵的。大部分的我们都认同生活幸福的观点，而身体健康其实是重要组成成分。从小到现在，书本上的教导，老师的教诲（这里特指海容老师）都告诉我们，身体是承载一切的容器，人没了，其他一切无意义。我一直十分认同这样的观点，但常常也只停留在认同了，大概是没有过大病。这次的我有了胃病，细菌感染，虽非大病，但也给我上了一课。年轻的身体也许经得起几次折损，不会有大问题，可也许会留下难以消除的病根，触目惊心的新伤疤历历在目。

上星期和母亲打了电话，我问她讲，别累着自己，少赚几个钱也没什么，健康最重要。不太明亮的灯下，我看到了她带着皱纹的苦笑，又是熟悉的那一句：我们不赚钱，怎么生活嘛...... 我还是带着笑劝着她多休息好，全然不敢提我的病。俗话说，报喜不报忧，而我既不报喜也不报忧。我始终觉得，有结果才有底气说，未成之事不敢言。这是高一时一位外校英语老师留给我最深刻的一句话。

经常和同学带着笑说，不知不觉马上大三了。时光一直在加速，这是一个一生都要讨论的话题。所以人也变得理所应当，大二下，大三上，大三下，就要结束了。紧迫的时间线让人想准备充分一些，多些比赛，多些项目......想好好迎接明年的夏令营。

忙碌利弊皆有。忙碌使我对时间的利用相对更充分了，每日没有太多伤春悲秋的闲暇时光，一直赶着往前走，往前走，换句话来说就是充实的。另一方面，我也发现了，一直处在奔走状态的我，有时过于疲惫，弦绷得过紧，容易断，张弛有度无疑是最好的。而且参加太多比赛和项目其实并不是一件好事，每个人的承受能力有限，还要兼顾课内的学习任务。大二的课本本身就相对难一些，我似乎已经快接近这个极限了，不敢再续，剩下的日子还是要学好课内。

入校后的我一个人过得一直很好，也很喜欢一个人埋在自己的世界里。可能是之前接触了巧芸等人的不婚主义，我开始渐渐理解了，一个人的世界会少去了很多烦恼，但是，我还是明白，可能几年后，或十几年后，我还是会结婚生了，遵从世俗的安排。这是被观念限制住的我，也是被父母、被祖辈束缚住的我。

一个人吃饭，一个人自习，一个人发愣。我觉得我是习惯独处的，喜欢独处的。独处不代表不与人交往，而是能安然自若地享受一个人的生活，客观地看，我是一个比较能独处的人，能够沉下心写字，看书。但我又不完全属于独处的人，我会被一些关系所影响，偶也感到孤独，当有倾诉对象时，我会好好抓住机会，发泄或分享。这也许对我的交往观有很大的教导，相处久了，才会知心。但往往这个过程总是折磨的，你会思来想去，苦恼不已。如若离高的聊天其实对双方都不好。想要打破，但又有理由万千阻碍着。最重要的还是，还是不想失去这个朋友。

自己思考，不如多听听他人的观点，看书或视频都是途径。很多时候，你自己模棱不定，需要几言点醒。很多核心的问题需要自己明晰。恋爱观这个问题不重要，也很重要。 其实不用太在意，没你想得那么真情。信我。

网上的一句话："我其实动心了，但赶路要紧，我没说"

说到我心坎里去了，我相信也说到很多人心坎里了。赶路？停下驻足？我还回答不了，可能没有回答吧。就像苏格拉底让弟子们找麦穗一样，有的人希望多等一会，找到最好的，但最好的已经过去，不可再来；有的人看见满意的，就摘下，后面其实有更好的。无论如何种选择其实都是你的人生，没有最好。

希望我能在现实中一点点践行，坦然接受。

且行且止，张弛有度。我想看到自信阳光的自己。

（最后一点碎碎念，最近痴迷于《我记得》，我这辈子又是谁的教父或伴侣）
