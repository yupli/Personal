# 人工智能

本分区用于 **机器学习、深度学习、大模型应用与工程实践** 相关笔记：既可记理论提纲，也可记复现步骤、数据与评测。

::: tip 写作建议
优先写清「目标 → 方法 → 输入输出 → 坑点」，并附上关键引用或链接；长系列可用多篇连载并在侧栏中排序。
:::

## 建议子主题（可按需建页）

- 经典机器学习与统计学习笔记
- 深度学习基础（优化、正则、架构）
- 自然语言 / 多模态应用速记
- 训练与推理工程（环境、加速、部署）

## 新建一篇笔记

1. 新建文件，例如 `docs/ai/transformer-notes.md`。
2. 在 `docs/.vitepress/config.mts` 的 `/ai/` 侧栏中加入：

```ts
{ text: "Transformer 备忘", link: "/ai/transformer-notes" },
```

保存后本地预览即可看到侧栏入口。
