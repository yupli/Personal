# Dijkstra 算法

**Dijkstra 算法**在带权有向图（或无向图）上求解**单源最短路**：给定源点 $s$，求到各顶点的最短距离。当**边权非负**时，算法正确；若存在负权边，需改用 **Bellman-Ford** 等。

---

## 1. 问题与记号

- 图 $G=(V,E)$，边权 $w(u,v) \ge 0$（$(u,v)\in E$）。  
- 源点 $s \in V$。  
- 目标：对所有 $v \in V$，求 $s$ 到 $v$ 的**最短路径长度** $\delta(s,v)$（或判为不可达）。

**基本思想**：维护已求出最短距离的**闭集** $S$，每次从 $V\setminus S$ 中选取当前**距离估计最小**的顶点加入 $S$，并用其**松弛**邻边（贪心、非负权保证正确性）。

---

## 2. 伪代码（与优先队列）

```
输入: G, w, s
初始化: dist[s] ← 0；其余 dist ← +∞；S ← ∅；Q ← 最小堆，键为 dist

将 s 插入 Q

当 Q 非空:
    从 Q 取出 dist 最小的顶点 u
    若 u 已在 S 中则继续（懒惰删除时）
    将 u 加入 S
    对每条边 (u,v) ∈ E:
        若 dist[u] + w(u,v) < dist[v]:
            dist[v] ← dist[u] + w(u,v)
            在 Q 中更新或插入 v
返回 dist
```

- 用**二叉堆**时，典型的单操作为 $O(\log |V|)$，总复杂度约 **$O((|V|+|E|)\log |V|)$**。  
- **斐波那契堆**可改进至 $O(|E| + |V|\log|V|)$（理论意义为主）。

---

## 3. 正确性要点

- **非负权**保证：一旦某顶点 $u$ 以当前最小 dist 从堆中**确定**，其 dist 即为 $\delta(s,u)$，不会再被缩短。  
- 若存在**负权边**，该性质不成立，Dijkstra **可能错误**，应使用 Bellman-Ford 或 Johnson 等。

---

## 4. 应用与扩展

- **路网络、路由、资源分配**中的最短路。  
- 需**完整最短路树**时，在松弛时记录 `parent[v]←u`。  
- **全源最短路**且非负：可对每个源点跑一次 Dijkstra；或 Floyd（稠密小图）等。
