分区说明

本页目录

运筹优化 ​

本分区用于整理 运筹学、优化建模、算法与仿真 等笔记（线性 / 整数规划、列生成、启发式、库存与物流优化等均可按需拆页）。

写作建议

每篇笔记尽量包含：问题定义 → 模型或算法要点 → 可复用结论 / 伪代码 → 参考或延伸阅读。公式可直接使用 Markdown 数学扩展（需时在主题中接入 KaTeX）。

建议子主题（可按需建页） ​

  • 线性规划与对偶、灵敏度
  • 整数规划与分支定界 / 割平面
  • 列生成 / Benders / 分解方法
  • 元启发式与邻域搜索
  • 仿真与离散事件建模（按需）

新建一篇笔记 ​

  1. 新建文件，例如 docs/or-opt/lp-basics.md。
  2. 在 docs/.vitepress/config.mts 的 /or-opt/ 侧栏中加入：
ts
{ text: "线性规划基础", link: "/or-opt/lp-basics" },
1

保存后本地预览即可看到侧栏入口。
